gobuild4

Download golang builded file directly.

Usage

Copy the whole script to shell to build and download for repo https://github.com/codeskyblue/grget

bash -c "$(curl -s http:///grins.sh)" grins.sh codeskyblue/grget

Another way to do that

$ go get -v github.com/codeskyblue/grget/grcli
$ grcli install codeskyblue/grget
