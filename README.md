# grget
very simple go online build.

## Limits
* The repo must be in github.com
* Must be golang repo
* Repo must have folder `Godeps` in top dir.

## Usage

```
curl http://grget.shengxiang.me/codeskyblue/minicdn/master/linux/amd64 -o minicdn
```
